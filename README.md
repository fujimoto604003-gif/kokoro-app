# Mom’s ココロ Diary

ママのためのメンタルケア＆日記アプリです。

## 概要

日々の出来事（日記）と、心の状態（VARY Parenting Scaleに基づく25の質問）を記録し、
レーダーチャートで心のバランスを可視化します。
また、状態に応じたキャラクターからのアドバイスや、詳細な診断メッセージが表示されます。

## 機能

- **日記記録**: 今日の出来事と気分を一言で記録。
- **心の診断**: 25の質問に答えて、7つの領域（愛着、効力感など）のバランスを診断。
- **可視化**: レーダーチャートで直感的に状態を把握。
- **履歴**: 過去の記録を一覧表示し、詳細を振り返り可能。

## ローカルでの実行方法

```bash
# 必要なライブラリのインストール
pip install -r requirements.txt

# アプリの起動
streamlit run app.py
```

## 使用技術

- Python
- Streamlit
- Plotly (可視化)
- SQLite (データ保存)
